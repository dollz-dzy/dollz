Despre Start_view

Este primul fisier .bat aparut pe site acesta permite userilor prin cmd sa mearga in anumite programe/foldere pentru a accesa info si programe pentru start

Precum folderul /startup si alte programe windows
Descarcare
Puteti instala programul prin urmatoarele link-uri
Mega (2kb)
Codul sursa
Acesta este un program mic deci codul sursa este disponibil
::=============================== :: :: Dollz_dzy pe github :: start_view.bat :: ::=============================== @echo off :: Colectam cateva variabile set username=%USERNAME% :meniu cls echo Bun venit %username%! echo. echo [ exit ] Exit [ info ] Info echo. echo [ 1 ] Deschide taskmgr echo [ 2 ] Deschide folderul startup echo [ 3 ] Deschide msconfig echo [ 4 ] Deschide services.msc echo [ 5 ] Deschide taskschd.msc echo. :: Setam pormptul set /p input=[ %username% ] : ::Aici is cateva conditi pentru fiecare alegere a userului if /i "%input%"=="1" ( cls start taskmgr goto meniu ) else if /i "%input%"=="2" ( cls start shell:startup goto meniu ) else if /i "%input%"=="3" ( cls start msconfig goto meniu ) else if /i "%input%"=="4" ( cls start services.msc goto meniu ) else if /i "%input%"=="5" ( cls start taskschd.msc goto meniu ) else if /i "%input%"=="exit" ( cls echo Scriptul se inchide.La revedere %username%! timeout /t 2 >nul exit /b ) else if /i "%input%"=="info" ( cls echo [ Info ] [ version 1.0 ] echo. echo Bun venit pe start_view unde puteti accesa diferite locati/programe legate de startup. echo Sper ca va fi de folos. echo. pause goto meniu ) else ( cls echo Comanda necunoscuta. echo. echo Daca nu intelegi cum functioneaza teas ruga sa alegi o optiune din [] precum [ 1 ] sau [ exit ]. pause goto meniu ) :: facut in 4/26/2025
Codul sursa poate uneori sa apara ciudat ,deoarace acesta nu e destinat pentru a fi download-dat si vizualizat doar.
